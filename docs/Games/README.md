# Game Design Guidelines

This document outlines the overall direction which all games should follow.

## Basic Structure

Every game has two screens:

- **Global Screen** — The shared display (TV/projector) that everyone watches together. Renders the main game action.
- **Controller Screen** — Each player's phone. Provides input controls (buttons, joystick, motion). May also show player-specific info.

The global screen is the primary visual output. Controller screens should be minimal and focused on input.

## Game Feel Guidelines

- Games should be instantly understandable — no tutorial needed
- Rounds should be short (1-5 minutes) to keep the party moving
- Visual feedback should be clear on the global screen
- Controller should provide haptic feedback (vibration) on important events
- Support 2-4 players minimum; games should handle uneven player counts gracefully
