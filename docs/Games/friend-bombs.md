# Friend Bombs Game Concept

A 2D platformer battle game for 2-4 players, inspired by Super Smash Bros mechanics.

## Core Mechanics

- Players fight on a single 2D platform arena
- Items (bombs, spikes, bubblegum, artifacts) fall from the sky
- Players pick up items and throw them at opponents
- **HP system** (Smash-style): lower HP = higher knockback from attacks
- Last player standing wins the round

## Controls

- **Joystick** — Move left/right, jump
- **Buttons** — Pick up, throw, use item
- **Tilt** — Aim throw direction

## Items

| Item | Effect |
|------|--------|
| Bomb | Explodes on impact, high knockback |
| Spike | Pierces through obstacles |
| Bubblegum | Slows down hit player temporarily |
| Artifact | Scores bonus points (non-damaging) |

## Global Screen

- Side-view 2D arena with platforms
- Player HP bars displayed above each character
- Item spawn indicators
- Kill feed and score display
- Round timer

## Rounds

- Timed matches (2-3 minutes)
- Most KOs or last standing wins
- Sudden death if tied
