# Cook-Off Game Concept

A speed-cooking game inspired by simplified Papa's Pizzeria mechanics.

## Core Mechanics

- Each player must assemble a dish (burger, pizza, etc.) following orders
- Orders appear with specific ingredient requirements
- Players grab ingredients, prepare them, and assemble in the correct order
- Speed and accuracy determine score

## Controls

- **Tap** — Select ingredients, interact with stations
- **Swipe** — Move between stations (chopping, grilling, plating)
- **Drag** — Assemble ingredients on the plate

## Global Screen

- Shows each player's station side-by-side
- Current order ticket for each player
- Score and progress bar
- Time remaining in the round

## Rounds

- Multiple rounds with increasing complexity
- More ingredients, stricter time limits
- Bonus points for perfect orders
- Elimination-style: lowest scorer each round is eliminated
