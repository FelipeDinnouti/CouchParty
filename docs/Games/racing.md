# Racing Game Concept

A simple top-down or side-view racing game for 2-4 players.

## Core Mechanics

- Players race from point A to point B (or complete laps)
- Each player controls their vehicle via phone
- First to cross the finish line wins

## Controls

- **Joystick** — Steer left/right, accelerate/decelerate
- **Tilt** — Optional: tilt phone to steer

## Global Screen

- Top-down view showing all racers
- Course with obstacles, shortcuts, and hazards
- Lap/position display for each player
- Finish line with celebration animation

## Implementation Notes

- Keep physics simple (arcade-style, not simulation)
- Power-ups: speed boost, oil slick, shield
- 3-5 laps per race
- Track obstacles that reset each lap
