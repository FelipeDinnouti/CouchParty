## Basic Structure

Every game has two screens, the global screen and the controller screen. The controller screen is the one that each player holds and has access to. Most of the times, it can be simply just controllers, or it can show something else more relevant. In theory, the controller screen has greater 2D/3D rendering capabilities, but probably it won't be used that way often, since the global screen is the one that shows the content for all to see together.
